# Python Handbook

A comprehensive Python learning and reference repository covering AI/ML, finance, networking, and LLM development.

## Project Structure

- AI & Machine Learning (`ai/`)
- Finance (`finance/`)
- Network (`network/`)
- LLM (`llm/`)
- Library (`library/`)

## Command

**Install**

```sh
uv init
uv venv
source .venv/bin/activate
uv add scikit-learn numpy matplotlib
```

**Activate**

```sh
uv venv
source .venv/bin/activate
uv sync
```
